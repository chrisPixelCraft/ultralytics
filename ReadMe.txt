"afterlabel/"
這資料夾是延續crosslabel資料夾的，裡面有對前面的標註做一些後處理，讓結果比較符合實際狀況。
剩下自己研究。

"camera_calibration/"
這裡面是利用chess board calibration來校正我們所使用的相機，當初的相機都是租的，所以要靠當初拍攝的棋盤格照片來校正。
程式碼已經附上了，詳細原理請自行研究，好好上電腦視覺。

"crosslabel/"
這資料夾是延續label資料夾的，主要是為了跨相機的標註及資料分析。
詳細內容我已經忘光了，裡面的程式碼都不難，請自己研究，關於label的方法設計也請自己想想，可以拿我這裡當作參考就好。


"GUI/"
這個GUI是用來簡單label的，並不是最終版的label工具，僅供參考。
正版的label工具已經遺失了，可能要請自己去挖，那些是我當初從網路上的工具載下來並從程式碼去更改，改成適合我們使用的樣子。
如果還需要用的話，請自己去研究。

"homo/"
這裡面是用來做extrinsic calibration的相關資料，詳細狀況請好好去上電腦視覺課。

"label/"
這是當初要為了label使用到的資料夾，不過現在應該用不到了，就不做太多介紹。
在ByteTrack資料夾中有初步跑出來的結果，還有一些統計出的方框數量。當初是為了讓工讀生比較好標，先用現成的演算法跑出第一版結果。
在gatherNzip資料夾的內容是為了打包成給工讀生用的資料包，已經寫好script了。
mask中是關於ROI的產生，同時也有一些程式碼用來過濾不再範圍內的方框。
text則是根據BYTE的結果產生LABEL用的資訊。


"video/"
這裡面的程式碼是我一開始處理蒐集的影片時用到的一些程式碼，留下來供參考。
有一些像是轉檔、同步、拆分圖片、更改fps、取得影片資訊等功能，請自行研究。
